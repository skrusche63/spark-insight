Real-time Insights with Apache Spark

Association Rule (AR) Mining

Scala and Apache Spark based implementation of most recent Top-K Association Rule Algorithms as proposed by Philippe-Fournier Viger. Top-K Association Rule Mining overcomes one of the major problems in traditional Association Rule approaches, i.e. tuning the minimum support parameter.

Top-K Association Rule Mining therefore is an appropriate candidate for applying this technique to real-time data streams. Read more.
